1. When should accessor methods be used?
Answer: When we want to access the value of a property of an object from outside that object, but do not want to allow outsiders to change the value of that property

2. If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
Answer: Yes, in Java we can create different constructors for different purpose, as long as they have different paramenters
